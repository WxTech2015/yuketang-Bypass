## 📘 使用教程

### 1️⃣ 安装 Node.js

请前往 Node.js 官方网站下载安装：
👉 [https://nodejs.org/zh-cn/zh/download](https://nodejs.org/zh-cn/zh/download)

### 2️⃣ 下载本项目

将项目文件克隆或下载到本地。

### 3️⃣ 启动程序

在项目目录下执行以下命令：

```bash
npm start
# 或
node index.js
```
